# Mirage
# ESP32-C3 Wi-Fi Device Controller

## Overview
This project is designed to control devices through Wi-Fi using the ESP32-C3 microcontroller. The project includes both the hardware design (PCB) and the software (firmware) to enable Wi-Fi-based control of various devices.

## Features
- **Wi-Fi Connectivity**: Connect and control devices over a Wi-Fi network.
- **Web Interface**: Simple web-based interface for device control.
- **Open Source**: Both hardware and software are open-source.
